# Privacy Policy — Maxxy

_Last updated: June 2026_

Maxxy ("the bot") is a Discord accountability bot used within this server to help members set goals, post progress check-ins, and stay accountable to one another. This page explains what information Maxxy collects, how it's used, and how you can request changes or removal.

## What Maxxy collects

When you use Maxxy's commands, it stores the following information:

- **Your Discord user ID and username** — used to identify which goals belong to you and to tag you in reminders.
- **Goal information** — the title, description ("what does success look like"), and optional timeline you provide when you run `/goal-start`.
- **Check-in content** — the progress update text you submit with `/checkin`, along with the timestamp of each check-in.
- **Screenshot attachments** — if you attach a screenshot to a `/checkin`, Maxxy stores a link to that file (hosted by Discord itself, not by Maxxy).
- **Event history** — a timestamped record of actions on your goals (started, paused, resumed, verified, achieved, abandoned, and reminders sent), used to build your goal timeline and to determine when reminders should be sent.

Maxxy does not collect your email address, IP address, payment information, or any data outside of what you explicitly submit through its commands.

## How this information is used

- To track and display your active goals (`/my-goals`, `/active-goals`)
- To post goal updates and check-ins in the designated goal-tracking channel, visible to other server members
- To send reminder messages (publicly in the goal-tracking channel, privately via direct message, or as an alert to admins) if a goal goes a number of days without a check-in
- To let Senior, admin, and TAMBAY roles verify check-ins and mark goals as achieved

## Where this information is stored

Goal and check-in data is stored in a private database that only the bot's operator can access directly. It is not sold, shared with advertisers, or provided to any third party outside of normal Discord functionality (e.g. messages Maxxy posts are visible to anyone with access to the relevant channel, the same as any other message in this server).

## Visibility within the server

By design, goal announcements and check-ins are posted publicly in the server's goal-tracking channel as part of how the accountability system works. If you'd prefer not to have your goal progress visible to other members, this bot may not be the right fit for your use case — please reach out to a server admin with questions before using it.

## Your choices

- You can pause (`/goal-pause`) or abandon (`/goal-abandon`) a goal at any time.
- You can request that a server admin delete your goal and associated data from Maxxy's database by contacting them directly.
- If you leave the server, your existing goal data remains in the database unless an admin removes it on request.

## Changes to this policy

This policy may be updated as Maxxy's features change. Continued use of the bot after changes are posted here constitutes acceptance of the updated policy.

## Contact

Questions about this policy or requests to access/delete your data can be directed to a server admin or Senior.
